# show-tracker

## Project Overview
A personal show tracker designed to help users manage their entertainment consumption. This application allows users to catalog shows, track viewing status, and maintain personal ratings and notes.  

## Key Features
- Dynamic Catalog: Browse and search your show collection with persisted states for seamless navigation.
- Status Tracking: Organize shows into Want, Active, Done, or Dropped categories with ease.
- Customizable UI: Personalize your experience with light/dark mode toggles.
- Interactive Details: Easily update show statuses, provide 1–5 star ratings, and add personal commentary for every entry.
